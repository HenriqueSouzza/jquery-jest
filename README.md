# Jquery with Jest

Projeto para apresentar um pequeno exemplo de como usamos Jest com Jquery.

## Requirement

- Node 20.10
- Yarn

## Setup

clona o projeto

```bash
git clone git@github.com:HenriqueSouzza/jquery-jest.git
```

Abrir pasta

```bash
cd jquery-jest
```

Instalar as dependências

```bash
Yarn install
```

## How Usage

Rodar teste básico

```bash
Yarn test
```

Rodar teste com report coverage

```bash
Yarn test-coverage
```
